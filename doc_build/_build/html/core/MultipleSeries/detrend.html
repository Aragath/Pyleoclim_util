MultipleSeries.detrend (pyleoclim.core.ui.MultipleSeries.detrend)¶

pyleoclim.core.ui.MultipleSeries.detrend(self, method='emd', **kwargs)¶

Detrend timeseries

Parameters

  • method (str, optional) –

    The method for detrending. The default is ‘emd’. Options include:

    • linear: the result of a linear least-squares fit to y is subtracted from y.

    • constant: only the mean of data is subtrated.

    • ”savitzky-golay”, y is filtered using the Savitzky-Golay filters and the resulting filtered series is subtracted from y.

    • ”emd” (default): Empirical mode decomposition. The last mode is assumed to be the trend and removed from the series

  • **kwargs (dict) – Relevant arguments for each of the methods.

Returns

new – The detrended timeseries

Return type

pyleoclim.MultipleSeries
