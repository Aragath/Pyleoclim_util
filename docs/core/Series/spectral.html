Series.spectral (pyleoclim.core.ui.Series.spectral)¶

pyleoclim.core.ui.Series.spectral(self, method='wwz', freq_method='log', freq_kwargs=None, settings=None, label=None, verbose=False)¶

Perform spectral analysis on the timeseries

Parameters

  • method (str) – {‘wwz’, ‘mtm’, ‘lomb_scargle’, ‘welch’, ‘periodogram’}

  • freq_method (str) – {‘log’,’scale’, ‘nfft’, ‘lomb_scargle’, ‘welch’}

  • freq_kwargs (dict) – Arguments for frequency vector

  • settings (dict) – Arguments for the specific spectral method

  • label (str) – Label for the PSD object

  • verbose ({True, False}) –

Returns

psd – A PSD object

Return type

pyleoclim.Psd

Examples

Calculate the spectrum of SOI using the various methods and compute significance

In [1]: import pyleoclim as pyleo

In [2]: import pandas as pd

In [3]: from matplotlib import pyplot as plt

In [4]: data=pd.read_csv('https://raw.githubusercontent.com/LinkedEarth/Pyleoclim_util/Development/example_data/soi_data.csv',skiprows=0,header=1)

In [5]: time=data.iloc[:,1]

In [6]: value=data.iloc[:,2]

In [7]: ts=pyleo.Series(time=time,value=value,time_name='Year C.E', value_name='SOI', label='SOI')

# Standardize the time series
In [8]: ts_std=ts.standardize()

# WWZ
In [9]: psd_wwz=ts_std.spectral()

In [10]: psd_wwz_signif=psd_wwz.signif_test(number=10)

In [11]: fig,ax=psd_wwz_signif.plot(title='PSD using WWZ method')
<Figure size 1000x400 with 1 Axes>

In [12]: plt.close(fig)

#Periodogram
In [13]: ts_interp = ts_std.interp()

In [14]: psd_perio=ts_interp.spectral(method='periodogram')

In [15]: psd_perio_signif=psd_perio.signif_test()

In [16]: fig,ax=psd_perio_signif.plot(title='PSD using Periodogram method')
<Figure size 1000x400 with 1 Axes>

In [17]: plt.close(fig)

#Welch
In [18]: ts_interp = ts_std.interp()

In [19]: psd_welch=ts_interp.spectral(method='welch')

In [20]: psd_welch_signif=psd_welch.signif_test()

In [21]: fig,ax=psd_welch_signif.plot(title='PSD using Welch method')
<Figure size 1000x400 with 1 Axes>

In [22]: plt.close(fig)

#MTM
In [23]: ts_interp = ts_std.interp()

In [24]: psd_mtm=ts_interp.spectral(method='mtm')

In [25]: psd_mtm_signif=psd_mtm.signif_test()

In [26]: fig,ax=psd_mtm_signif.plot(title='PSD using MTM method')
<Figure size 1000x400 with 1 Axes>

In [27]: plt.close(fig)

#Lomb-Scargle
In [28]: psd_ls=ts_std.spectral(method='lomb_scargle')

In [29]: psd_ls_signif=psd_ls.signif_test()

In [30]: fig,ax=psd_ls_signif.plot(title='PSD using Lomb-Scargle method')
<Figure size 1000x400 with 1 Axes>

In [31]: plt.close(fig)
